Plugin que lista todos os posts de uma determinada categoria numa lista com marcadores ou numera&ccedil;&atilde;o em uma outra p&aacute;gina ou post - ou seja, cria uma esp&eacute;cie de categoria numa p&aacute;gina ou post -, personalizavel e bem simples de trabalhar.

== Description ==

* Cria links para posts na lista automaticamente;
* Pode ser personalizado;
* Pode-se escolher entre marcadores e n&uacute;meros;
* Inser&ccedil;&atilde;o pelo post, p&aacute;gina ou template PHP;
* Ordem por data (crescente ou decrescente) ou alfab&eacute;tica.

== Installation ==

* 1 - Fa&ccedil;a o download do plugin;
* 2 - Efetue upload do zip para o servidor na pasta de plugins;
* 3 - Ative o plugin no painel de controle;
* 4 - Configure-o em op&ccedil;&otilde;es.
